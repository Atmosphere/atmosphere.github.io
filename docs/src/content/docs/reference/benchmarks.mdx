# Performance Benchmarks

Atmosphere runs JMH micro-benchmarks on every push to `main` that touches performance-sensitive code, plus weekly scheduled runs. Results are tracked over time to detect regressions automatically.

## What We Measure

<CardGrid>
  <Card title="CheckpointStore" icon="rocket">
    `save`, `load`, `fork`, `list` throughput at 100 / 1K / 10K snapshots.
    Validates that durable workflow state doesn't become a bottleneck.
  </Card>
  <Card title="Broadcaster Dispatch" icon="rocket">
    Broadcast-to-N-subscribers throughput at 1 / 10 / 100 / 1K subscribers.
    Measures the per-subscriber dispatch overhead in `DefaultBroadcaster`.
  </Card>
  <Card title="Coordinator Fan-Out" icon="rocket">
    Parallel agent dispatch latency at 2 / 4 / 8 / 16 agents.
    Validates virtual-thread-based fleet coordination scales linearly.
  </Card>
  <Card title="AI Interceptor Chain" icon="rocket">
    Pre/post-process chain traversal at 0 / 1 / 4 / 16 / 64 interceptors.
    Measures guardrail + memory + cost-filter overhead per prompt.
  </Card>
  <Card title="AgentRuntime Resolver" icon="rocket">
    Cached resolution path (warm) for the `ServiceLoader`-based runtime lookup.
    Validates the hot path is effectively zero-cost.
  </Card>
  <Card title="VT Pinning Check" icon="rocket">
    JFR-based scan for `jdk.VirtualThreadPinned` events under benchmark load.
    Proves no `synchronized` blocks pin virtual threads to carriers.
  </Card>

<script is:inline>
(async function() {
  const loading = document.getElementById('benchmark-loading');
  const canvas = document.getElementById('benchmark-chart');
  const tableDiv = document.getElementById('benchmark-table');

  try {
    // Fetch benchmark data from the benchmark-data branch
    const resp = await fetch(
      'https://raw.githubusercontent.com/Atmosphere/atmosphere/benchmark-data/dev/bench/data.js'
    );
    if (!resp.ok) {
      loading.textContent = 'No benchmark data available yet — the first CI run will populate this page.';
      return;
    }

    // data.js defines window.BENCHMARK_DATA
    const text = await resp.text();
    const fn = new Function(text + '; return window.BENCHMARK_DATA;');
    const data = fn();

    if (!data || !data.entries) {
      loading.textContent = 'Benchmark data format not recognized.';
      return;
    }

    // Build table with latest results
    const entries = data.entries['Atmosphere JMH Benchmarks'] || [];
    if (entries.length === 0) {
      loading.textContent = 'No benchmark entries found yet.';
      return;
    }

    const latest = entries[entries.length - 1];
    const benches = latest.benches || [];

    let html = '<table><thead><tr><th>Benchmark</th><th>Value</th><th>Unit</th><th>±Error</th></tr></thead><tbody>';
    for (const b of benches) {
      const name = b.name.replace(/^org\.atmosphere\.benchmarks\.jmh\./, '');
      html += '<tr><td><code>' + name + '</code></td><td><strong>' + b.value.toFixed(2) + '</strong></td><td>' + b.unit + '</td><td>±' + (b.range || '?') + '</td></tr>';
    }
    html += '</tbody></table>';
    html += '<p style="margin-top:0.5em;font-size:0.85em;color:#666;">Run: ' + new Date(latest.date * 1000).toISOString().split('T')[0] + ' · Commit: <code>' + (latest.commit?.id?.slice(0,8) || '?') + '</code></p>';

    tableDiv.innerHTML = html;
    tableDiv.style.display = 'block';
    loading.style.display = 'none';

    // Build chart with history (last 20 runs)
    if (entries.length > 1 && typeof Chart !== 'undefined') {
      const recentEntries = entries.slice(-20);
      const labels = recentEntries.map(e => new Date(e.date * 1000).toLocaleDateString());
      const benchNames = [...new Set(recentEntries.flatMap(e => (e.benches || []).map(b => b.name)))];

      const colors = ['#2563eb','#dc2626','#16a34a','#ca8a04','#9333ea','#0891b2','#e11d48','#4f46e5'];
      const datasets = benchNames.map((name, i) => ({
        label: name.replace(/^org\.atmosphere\.benchmarks\.jmh\./, ''),
        data: recentEntries.map(e => {
          const b = (e.benches || []).find(x => x.name === name);
          return b ? b.value : null;
        }),
        borderColor: colors[i % colors.length],
        tension: 0.3,
        fill: false,
      }));

      canvas.style.display = 'block';
      new Chart(canvas, {
        type: 'line',
        data: { labels, datasets },
        options: {
          responsive: true,
          plugins: {
            legend: { position: 'bottom', labels: { font: { size: 11 } } },
            title: { display: true, text: 'Benchmark History (last 20 runs)' },
          },
          scales: { y: { beginAtZero: false } },
        },
      });
    }
  } catch (err) {
    loading.textContent = 'Could not load benchmark data: ' + err.message;
  }
})();
</script>

## Running Locally

```bash
# Build the benchmarks module (profile-gated)
./mvnw -Pperf -pl modules/benchmarks -am package -DskipTests

# Run all JMH micro-benchmarks
scripts/benchmarks/run-jmh.sh

# Run a single benchmark
java -jar modules/benchmarks/target/benchmarks.jar CheckpointStoreBenchmark

# Virtual thread pinning check
scripts/benchmarks/vt-pinning-check.sh

# Streaming load test (server must be running separately)
scripts/benchmarks/run-streaming-load.sh --clients 1000 --messages 50
```

## CI Regression Detection

The [Benchmarks workflow](https://github.com/Atmosphere/atmosphere/actions/workflows/benchmarks.yml) runs automatically on:
- Pushes to `main` touching `modules/cpr/`, `modules/ai/`, `modules/checkpoint/`, `modules/coordinator/`, or `modules/benchmarks/`
- Weekly schedule (Monday 6am UTC)
- Manual dispatch

Regression threshold: **15%**. PRs that degrade any benchmark by more than 15% receive an automatic comment.

## Methodology

- **Hardware:** GitHub Actions `ubuntu-latest` runners (2-4 vCPU, 7-16 GB RAM)
- **JDK:** Temurin 21 LTS
- **JMH settings (CI):** 1 fork, 3 warmup iterations, 5 measurement iterations
- **JMH settings (local):** 2 forks, 5 warmup iterations, 10 measurement iterations (default)
- Numbers reflect relative performance on shared CI hardware — use for regression tracking, not absolute claims
