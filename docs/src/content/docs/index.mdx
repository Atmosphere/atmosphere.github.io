<CardGrid stagger>
  <Card title="Real-Time Core" icon="rocket">
    `@ManagedService` with transport-agnostic Broadcaster pub/sub. WebSocket, SSE, long-polling, and gRPC — auto-negotiated per client. Rooms, presence, message history, and interceptors built in.
  </Card>
  <Card title="AI / LLM Platform" icon="star">
    `@Agent` with pluggable `AgentRuntime` — auto-detects nine runtimes: the built-in OpenAI-compatible client plus Spring AI, LangChain4j, Google ADK, Embabel, Koog, Semantic Kernel, AgentScope, and Spring AI Alibaba adapters. Tool calling, conversation memory, filters, cost metering, and multi-model routing.
  </Card>
  <Card title="Multi-Agent Orchestration" icon="puzzle">
    `@Coordinator` + `@Fleet` for multi-agent systems. Parallel fan-out, sequential pipelines, and weighted routing. MCP, A2A, and AG-UI protocols for agent interoperability.
  </Card>
  <Card title="24-Chapter Tutorial" icon="open-book">
    Build a complete real-time application from scratch. Covers transports, rooms, AI streaming, MCP, clustering, channels, and production deployment.
  </Card>
  <Card title="Spring Boot & Quarkus" icon="setting">
    First-class starters with auto-configuration, native image support, Micrometer metrics, and OpenTelemetry tracing. Tracks Spring Boot 4.0.6 and Quarkus 3.35.
